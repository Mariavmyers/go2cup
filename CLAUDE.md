# GO2CUP — Project Rules for Claude Code

GO2CUP (sportdgx.com) is a static HTML World Cup 2026 fan-travel hub by
MainEntry LLC, hosted on GitHub Pages.

## Always follow these rules

### Wording
- Always write "soccer," never "football."
- Hot takes must be constructive: a genuine compliment or fair criticism,
  never mean-spirited.
- Keep all scores and facts accurate.

### Visuals & style
- Use FlagCDN flag images for countries — NEVER emoji flags.
- Brand colors: blue #2f80ff, green #21c25a, red #ff4338, gold #f5c518,
  dark backgrounds #07180e and #0a2413.
- Fonts: Bebas Neue, Oswald, Inter.
- Always match the existing site styling — reuse current patterns and CSS.

### Git (PowerShell — important)
- Use SINGLE-LINE git commit messages only. Never multi-line.
- List what changed before pushing.

### Site structure
- News: sports-news.html   - Articles: articles/   - Hot takes: hot-takes.html
- Host-city pages: host-cities/ (e.g. host-cities/atlanta.html)
- Stay/affiliate directory: where-to-stay.html

### Affiliate rules
- Affiliate links go through Travelpayouts (marker 736515.worldcup26).
- Preferred partners: Tiqets, Klook, Go City, Viator (attractions);
  Aviasales (flights); Kiwitaxi (transfers); Yesim / Airalo (eSIM).
- NEVER promote resale of World Cup match tickets — point fans to the official
  FIFA.com/tickets instead. Do NOT use TicketNetwork.

### Workflow
- Propose the plan before large changes.
- Build reusable templates I can clone across the 16 host cities.
